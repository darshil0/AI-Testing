# Contributing to AI-Testing

We welcome contributions to the AI-Testing framework! Whether you're fixing a bug, adding a new feature, or improving documentation, your help is appreciated.

## Getting Started

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** to your local machine:
    ```bash
    git clone https://github.com/YOUR_USERNAME/AI-Testing.git
    ```
3.  **Create a new branch** for your changes:
    ```bash
    git checkout -b feature/your-feature-name
    ```
4.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

## Making Changes

-   **Code Style**: We use `black` for code formatting and `flake8` for linting. Please ensure your code adheres to these standards.
-   **Commit Messages**: Write clear and descriptive commit messages.
-   **Tests**: If you add a new feature, please include tests.

## Submitting a Pull Request

1.  **Push your changes** to your fork:
    ```bash
    git push origin feature/your-feature-name
    ```
2.  **Open a pull request** on the main AI-Testing repository.
3.  **Provide a clear description** of your changes in the pull request.

Thank you for contributing!
